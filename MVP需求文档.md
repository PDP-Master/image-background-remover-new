### P0 必须实现

| 功能 | 描述 | 优先级 |

## 3. 技术架构

```
┌─────────────────┐      ┌──────────────────┐      ┌─────────────────┐
│   用户浏览器     │ ──▶  │  Cloudflare      │ ──▶  │   remove.bg    │
│  (HTML/CSS/JS)  │      │  Workers API      │      │   API           │
└─────────────────┘      └──────────────────┘      └─────────────────┘
```

### 技术选型

| 层级 | 方案 | 理由 |
|------|------|------|
| 前端 | HTML + JS | 轻量，无需后端 |
| 后端 | Cloudflare Workers | 免费额度足够，边缘计算 |
| 图片处理 | remove.bg API | 效果最好 |
| 部署 | Cloudflare Pages | 免费 + CDN 加速 |
| 存储 | 纯内存处理 | 不需要 |

---

## 4. 功能流程

```
上传图片 → 转Base64 → 发送给Worker → 调用remove.bg API → 返回结果 → 预览/下载
```
